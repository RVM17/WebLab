# Приложение-картотека

Небольшое веб-приложение для хранения списка студентов.
## Cтек

Python 3.10

БД: **SQLite**

Фронт: **HTML** + **JS**

Веб-сервер: **Flask**

## Запуск приложения

Для запуска сервера и клиента достаточно вызвать run.cmd и runclient.cmd, либо, используя IDE, запустить lab4.py и lab4client.py.
